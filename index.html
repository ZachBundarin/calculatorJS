Report

The calculator I have made is straight forward. Buttons are pressed and input the symbol into a string. When the equals sign is pressed the equation string is computated with the eval(expressionString) function. I used the two-part calculator with JavaScript guide in the bottom of the report to make this calculator. The main challenge was that the ‘+/-‘ button on the calculator worked extremely poorly. I tried the provided code but it would just add the entire string with a negative symbol which looked very bad. To resolve this issue I removed the ‘+/-‘ button all together since the ‘-‘ button works the same way with the eval(expressionString) function. This resolved most problems I had with the calculator.
